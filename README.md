# IMDb Data Analysis

## Project Description
This project analyzes IMDb Top 250 movies dataset using Python and Pandas.

## Objective
The goal of this project is to analyze the IMDb Top 250 movies dataset in order to explore patterns in movie popularity, genres, and runtime. 

This analysis aims to identify the most popular movies based on user votes, the most common genres, the average duration of movies, and the longest movie in the dataset.

## Analysis
- Top movie based on number of votes
- Most common genres
- Average runtime of movies
- Longest movie

## Tools Used
- Python
- Pandas

## Output
Excel file with summary of analysis (imdb_summary.xlsx)


## Codebook

- id: Unique identifier of the movie  
- title: Name of the movie  
- description: Short summary of the movie  
- content_rating: Age rating (e.g., PG, R)  
- budget: Budget of the movie (if available)  
- genres: List of genres associated with the movie  
- runtime_minutes: Duration of the movie in minutes  
- num_votes: Number of user votes (popularity indicator)
